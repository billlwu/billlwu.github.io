def f1(x):
    return x**2

def f2(x):
    return 1.1*x**2 + (x**1.5+10)*np.sin(10*x+1.5) + 30

def g(x):
    return (1 + np.sin(x/200 + 1.5)) * x ** 2 + x ** 1.5 + 10

def h(x):
    return (1 + np.sin(x / 200)) * x ** 1.5 + x ** 1.4 + 10

Using asymptotic notion, we write $f_2(x) = \Theta(x^2)$. What we really mean is $f_2 \in \Theta(x^2)$, since $\Theta(x^2)$ is a set of functions. The formal definition is given in CLSR.

Asymptotic notation allows us to reduce complex functional forms.
